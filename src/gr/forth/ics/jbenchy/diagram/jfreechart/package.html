Provides support for generating JFreeChart diagrams from reports generated by {@link gr.forth.ics.jbenchy.Aggregator}.

Package Specification

JFreeChart is a Java library that specializes in creating charts. These can be drawn on a {@link java.awt.Graphics} object, stored to a file (for example, as a JPEG), or embedded in a Swing application.

To see an example of this package expected usage, assume this preparation exists (which is explained in {@link gr.forth.ics.jbenchy.Aggregator}): 

Aggregator aggregator = ...;
Records records = aggregator. ... report(...);
Diagram diagram = DiagramFactory.newDiagram(records)
    .withTitle("My Title")
    .withRangeLabel("Time (milliseconds)");
The root access point of this package is {@link gr.forth.ics.jbenchy.diagram.jfreechart.ChartFactory}. Using this, a factory that can create various types of JFreeChart charts is created. 
ChartFactory chartFactory = new ChartFactory(diagram);
For example, to create a bar chart: 
Chart chart = chartFactory.newBarChart();
Or to create a line chart: 
Chart chart = chartFactory.newLineChart();
A {@link gr.forth.ics.jbenchy.diagram.jfreechart.Chart} object offers methods to {@link gr.forth.ics.jbenchy.diagram.jfreechart.Chart#newImage(int, int) an image of the chart}, to {@link gr.forth.ics.jbenchy.diagram.jfreechart.Chart#paint(java.awt.Graphics2D, double, double) paint} the chart to a {@link java.awt.Graphics} object, to {@link gr.forth.ics.jbenchy.diagram.jfreechart.Chart#write(int, int, String, java.io.File) save the chart into a file}, to {@link gr.forth.ics.jbenchy.diagram.jfreechart.Chart#newPanel() create a panel} that can be embedded in a Swing application. See the respective javadocs for a complete specification.

For a simple example, this would be used to create a jpg from the chart: 

chart.write(800, 600, "jpeg", new File("myDiagram.jpg"));

To create a panel that can be embedded in a Swing container: 

ChartPanel panel = chart.newPanel();
