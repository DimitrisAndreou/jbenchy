Provides core support for generating diagrams, typically from reports created by an {@link gr.forth.ics.jbenchy.Aggregator}.

Package Specification

The data set that supports a diagram is abstracted by the {@link gr.forth.ics.jbenchy.diagram.Diagram} interface.

See {@link gr.forth.ics.jbenchy.diagram.DiagramFactory} for API to create Diagram instances from an Aggregator report. See subpackages for supported renderings of this abstraction to actual diagrams.
